# CMake 快速开始指南

欢迎学习 CMake！这是一个为 C/C++ 程序员设计的交互式课程。

## 🎯 开始之前

### 确认环境

**Windows:**
```powershell
# 检查 CMake 是否安装
cmake --version

# 检查编译器（需要 Visual Studio 或 MinGW）
```

**WSL Ubuntu:**
```bash
# 安装 CMake 和编译器
sudo apt update
sudo apt install cmake g++ build-essential

# 检查版本
cmake --version
g++ --version
```

## 🛠️ 常见问题

### Q1: CMake 找不到编译器

**Windows:**
- 安装 Visual Studio（包含 MSVC 编译器）
- 或安装 MinGW

**Linux:**
```bash
sudo apt install build-essential
```

### Q2: 编译错误 "No CMAKE_CXX_COMPILER could be found"

需要安装 C++ 编译器。

### Q3: 构建目录很乱怎么办？

删除 build 目录重新开始：
```bash
rm -rf build  # Linux
rmdir /s build  # Windows CMD
Remove-Item -Recurse build  # PowerShell
```

### Q4: Windows 和 Linux 的区别？

主要在路径分隔符和生成器：
- Windows: `\` 和 Visual Studio 或 NMake
- Linux: `/` 和 Make

CMake 会自动处理，但最好两边都试试。
